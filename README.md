Laundry Wallah / Laundry Mart

This project is a simple website created using HTML.
It provides basic information about laundry services and allows users to book their laundry online.

Features

Displays available laundry services

Provides details like washing, ironing, and dry cleaning

Contains a simple booking form for customers

Includes contact information section

Technologies Used

HTML

How to Run

Open the index.html file in any web browser.

Explore the details about the laundry services.

Fill out the booking form if available.
